Sudoku Solver Personal Library

Express Apps

Express, Node.js and MongoDB allow the developer to program across the full stack using only one language: Javascript.

Sudoku Solver is an Express/Node.js application that can solve Sudoku problems and validate number placement. The application incorporates unit and functional tests to assure functionality through Chai and Mocha.

A small Express/Node.js application called Personal Library uses MongoDB to store and catalog previously-read books and comments. It utilizes API endpoint requests as well as front-end form submission to query the database and make changes.

Pomodoro Clock Drum Machine

React Apps

Two interactive React apps highlighting client-side engagement through audio and responsive UI.

Pomodoro Clock is an aide useful for applying the Pomodoro technique, a time management method for productive pursuits. The app consists of an alternating session timer and break timer including adjustable periods and audio clues upon timeout.

The second app, Drum Machine, combines click and keyboard events to trigger individual drum beats. Not as productive as Pomodoro timers but much more fun. Try combining different sequences to make a little beat!

Heat Map Choropleth Map

D3 Graphs

The D3 library is a Javascript library for manipulating objects based on data. This is especially useful for generating beautiful SVG graphs and maps.

I have created two SVG maps using data gathered from APIs: a heat map showing land-surface base temperature fluctuations over time as well as a choropleth map illustrating educational attainment by county in the United States.

About

I am Jacob Swartzentruber, a self-taught Javascript developer interested in exploring, implementing and learning more about web development solutions across the full-stack.

Procedural Terrain Generation

From high school through college, my personal projects evolved from creating simple 2D ball-bouncing simulations in C++ to more complex procedural generated landscapes and browser-based games in Java. Eventually, I began porting my games and simulations into Javascript for use in the browser which spurred me to explore web applications. When I discovered Javascript could be used across the full-stack, I immediately knew this is what I want to do professionally - creating interactive experiences for others that can be appreciated in tangible ways.

3d Printed Miniatures

What keeps me curious and constantly learning is getting hands-on and being able to see something come alive at the end of a hard day's work. Whether I am creating portfolios and web applications professionally, designing my next 3d printing project or creating Dungeons and Dragons campaigns to be shared with friends in my free time, I most enjoy collaborating with others to bring life to a previously non-existent product. If I can share these experiences with other people, it makes the end result ten times more fulfilling.

If you are looking for a curious web developer to join your team, want to talk about the nerdy aspects of 3D printing or simply want to say hello, send an email my way. I would love to hear from you!
